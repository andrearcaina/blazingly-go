## File Structure

```plaintext
blazingly-go/
├── channels/                  # goes over go channels
│    └── main.go               # main.go file
├── collections/               # goes over arrays/slices/maps
│    └── main.go               # main.go file
├── composites/                # goes over structs/interfaces
│    └── src/                  # src folder for each go file
│         └── structs.go       # structs.go file
│         └── interfaces.go    # structs.go file
│    └── go.mod                # go dependencies
│    └── main.go               # main.go file
├── control_flow/              # goes over if/switch statements and for loops
│    └── main.go               # main.go file
├── functions/                 # goes over functions
│    └── main.go               # main.go file
├── goroutines/                # goes over goroutines
│    └── main.go               # main.go file
├── generics/                  # goes over generics and type handling
│    └── main.go               # main.go file
├── intro/                     # hello world!
│    └── main.go               # main.go file
├── notes/                     # goes over all my notes
│    └── *.md                  # goes through each topic here
├── pointers/                  # goes over pointers
│    └── main.go               # main.go file
├── unicode/                   # strings/runes/bytes/UTF-8
│    └── main.go               # main.go file
├── variables/                 # variable types and declarations
│    └── main.go               # main.go file
└── go.mod                     # go dependencies
└── README.md                  # this README
```
